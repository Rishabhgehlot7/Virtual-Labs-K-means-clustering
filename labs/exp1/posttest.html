K-means Clustering

1. The initialization step in the K-means Clustering algorithm involves randomly selecting ________ as initial cluster centroids.

a) Data points
b) Cluster indices
c) Cluster labels
d) Centroid coordinates

2. K-means Clustering is an unsupervised learning algorithm used for __________.

a) Classification
b) Regression
c) Dimensionality reduction
d) Clustering

3. In K-means Clustering, K refers to __________.

a) The number of clusters
b) The distance metric
c) The centroid initialization method
d) The number of iterations

4. Which step in the K-means algorithm involves reassigning data points to their closest centroids?

a) Centroid initialization
b) Cluster assignment
c) Centroid update
d) Convergence check

5. K-means Clustering is sensitive to __________.

a) Initial centroid placement
b) The number of dimensions in the data
c) Outliers in the data
d) All of the above
