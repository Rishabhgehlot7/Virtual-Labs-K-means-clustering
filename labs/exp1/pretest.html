K-means Clustering

1. In K-means Clustering, the number of clusters to be formed is denoted by the parameter ________.

a) K
b) C
c) N
d) P

2. The ________ is a measure used in K-means Clustering to calculate the similarity between a data point and a cluster centroid.

a) Euclidean distance
b) Manhattan distance
c) Hamming distance
d) Cosine similarity

3. K-means Clustering can be applied to which type(s) of data?

a) Numerical data
b) Categorical data
c) Both numerical and categorical data
d) Image data

4. The centroid in K-means Clustering represents __________.

a) The average of all data points in a cluster
b) The data point farthest from all other data points
c) The data point closest to all other data points
d) The median of all data points in a cluster

5. The K-means algorithm begins by __________.

a) Randomly initializing the cluster centroids
b) Assigning all data points to their closest centroids
c) Calculating the average distance between data points and centroids
d) Iteratively updating the cluster centroids
